# ⏰ Tid brukt på dokumentjaging

| Aktivitet | Timer/uke |
|-----------|-----------|
| Skrive forespørsler | 1-2 timer |
| Følge opp manglende svar | 2-3 timer |
| Purringer og telefonsamtaler | 1-2 timer |
| Sortere og arkivere vedlegg | 1-2 timer |

### **= 5-10 timer per uke**
*som kunne vært brukt på verdiskapende arbeid*

---

# Problemet

- 📧 E-post forsvinner i innboksen
- ❓ Kunden vet ikke hva du vil ha
- 🔄 Du må purre manuelt - igjen og igjen
- 📁 Dokumenter må sorteres og navngis manuelt
- 😤 Frustrerende for deg OG kunden

---

# Løsningen: Regnskapsbruker

En enkel plattform for dokumentutveksling mellom deg og kundene dine.

---

# Slik fungerer det

### For deg (regnskapsfører):

1. **Opprett forespørsel** → Velg mal, sett frist
2. **Kunden varsles** → Automatisk e-post
3. **Automatisk purring** → Systemet følger opp
4. **Dokument levert** → Du varsles, alt arkivert

### Tid brukt: **2 minutter** i stedet for 2 timer

---

# Slik fungerer det

### For kunden din:

1. **Får e-post** → "Du har en oppgave fra regnskapsfører"
2. **Logger inn** → Ser tydelig hva som trengs
3. **Laster opp** → Dra fil eller ta bilde med mobilen
4. **Ferdig** → Grønn hake, oppgave fullført

### Ingen forvirring, ingen glemte e-poster
