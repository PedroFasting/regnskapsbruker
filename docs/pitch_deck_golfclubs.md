# Golfklubber er spesielle

**Kompleks dokumentflyt:**
- 📊 Kontingentrapporter fra medlemssystem
- 🚜 Bilag fra greenkeeper (ofte i lommen)
- 🏆 Turneringsregnskap
- 🤝 Sponsoravtaler med motytelser
- 📋 Styreprotokoller og signaturer

**Flere involverte:**
- Daglig leder
- Greenkeeper  
- Styret (3-5 personer)
- Økonomiansvarlig

---

# ⏰ Typiske problemer

| Problem | Konsekvens |
|---------|------------|
| Greenkeeper har kvitteringer i lommen | Bilag blir våte, mistes, glemmes |
| GolfBox-rapport kommer sent | Avstemming forsinkes |
| Styret skal signere årsregnskap | Jager 5 signaturer i uker |
| Sesongsvingninger | Kaos vår/sommer, stille vinter |
| Mange inntektskilder | Greenfee, kiosk, proshop - vanskelig å avstemme |

---

# For styret - digital signering

Årsmøtet nærmer seg. Styret skal signere.

**Før:**
- Send PDF på e-post til alle 5
- Vent... vent... purr...
- "Kan du printe og scanne?"
- 3 uker går

**Etter:**
- Send signeringsforespørsel
- Alle signerer med BankID på mobilen
- Ferdig signert på 2 dager
- Klart for årsmøtet ✅

# Slik fungerer det

### For deg (regnskapsfører):

1. **Opprett forespørsel** → Velg mal for golfklubb
2. **Velg mottaker** → Daglig leder eller greenkeeper
3. **Automatisk purring** → Systemet følger opp
4. **Dokument levert** → Alt arkivert og klart

### Tid brukt: **2 minutter** i stedet for 2 timer

# Hvorfor golfklubber?

**~170 golfklubber i Norge**
- De fleste har ekstern regnskapsfører
- Kompleks dokumentflyt = stort behov
- Tydelig sesongprofil vi kan automatisere
- Fornøyde klubber → referanser til andre

> Løser vi det for golfklubber, løser vi det for 
> idrettslag generelt (fotball, tennis, ski...)

# Er dette noe for deg?

**Passer hvis du har:**
- Golfklubber (eller andre idrettslag) som kunder

- Du har 2+ golfklubber i porteføljen
- Du ønsker å styrke tilbudet mot idrett/foreninger
